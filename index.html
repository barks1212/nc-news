Northcoders News API

Articles

GET requests

  • /api/articles - Retrieve all articles
  • /api/articles/:articleid - Retrieve specific article
  • /api/articles/:articleid/comments - Retrieve specific article comments

POST requests

  • /api/articles/:articleid/comments - Post comment to article

PUT requests

  • /api/articles/:articleid?vote=up/down - increment article vote up or down

DELETE requests

  • /api/articles/:articleid - delete article

Comments

GET requests

  • /api/comments - get all comments
  • /api/comments/:commentid - get specific comment

PUT requests

  • /api/comments/:commentid?vote=up/down - increment comment vote up or down

DELETE requests

  • /api/comments/:commentid - Delete comment

Topics

GET requests

  • /api/topics - get all topics
  • /api/topics/:topic/articles - get all articles for specific topic

POST requests

  • /api/topics/:topic/articles - post article to specific topic

Users

GET requests

  • /api/users - get all users
  • /api/users/:username - get specific user
  • /api/users/:username/articles - get articles by user
  • /api/users/:username/comments - get comments by user
